## 🔧 Setup and Installation

### **1️⃣ Clone the Repository**  
```bash
git clone https://github.com/your-username/user-management-api.git  
cd user-management-api
```
### **2️⃣ Install Dependencies**
```bash
npm install
```
### **3️⃣ Start the Server**
```bash
npm run dev
```
